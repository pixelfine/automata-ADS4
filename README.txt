Le projet a été réalisé sur Eclipse en utilisant JavaFx en tant qu'interface graphique.
La classe Main est la classe qu'il faut Run.

Condition pour le Run : 
-Avoir JavaFx
-Si vous utilisez le programme via le terminal, il faudra compiler toute les classes avant 
 de le Run. Javac de toute les classe.
-Lors de la compilation vous aurez des scanners : 
 Voulez vous lancer l'interface graphique ? : 'yes' | 'no'
 Si vous répondez no : 
	Inserer le nom des fichiers et tapez STOP pour finir
 	Exemple de fichier : 
		programme.txt
		grille.txt
		STOP
 Si vous répondez yes :
	L'interface graphique se lancera.
